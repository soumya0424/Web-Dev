## Overview

Web-Dev is a collection of frontend web development lab exercises covering HTML structure and PHP basics. Organized into numbered lab folders for progressive skill-building.

### Why Web-Dev?

This repository documents structured lab-based learning through hands-on frontend exercises. The core topics include:

- 🏗️ **HTML Structure:** Semantic HTML, forms, tables, and page layout.
- 🔧 **PHP Basics:** Server-side scripting and dynamic page generation.
- 📚 **Lab Organization:** Six numbered lab folders with progressive exercises.
- 💻 **Hands-On Practice:** Real exercises from web development coursework.

## Getting Started

### Prerequisites

This project requires the following:

- **Languages:** HTML, PHP
- **Server:** Any local PHP server (e.g., XAMPP, WAMP)

### Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/soumya0424/Web-Dev
    cd Web-Dev
    ```

### Usage

Open HTML files directly in a browser, or run PHP files through a local server.
